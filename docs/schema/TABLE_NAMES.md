# テーブル名一覧（日本語名付き）

このドキュメントは、データベース内のすべてのテーブルとその日本語名をまとめたものです。

## アプリケーション固有テーブル

| テーブル名（英語） | テーブル名（日本語） | 説明 |
| ---- | ---- | ---- |
| `users` | ユーザー | ユーザー情報（名前、メールアドレス、認証情報など） |
| `stores` | 店舗 | 店舗情報（店名、住所、営業時間、おすすめのお酒タイプなど） |
| `diagnose_results` | 診断結果 | お酒診断の結果（診断タイプ、候補一覧など） |
| `diagnose_feedbacks` | 診断フィードバック | 診断結果へのフィードバック（1-5点の評価、コメントなど） |
| `user_diagnose_results` | ユーザー診断結果 | ユーザーと診断結果の紐付け（中間テーブル） |
| `user_visited_stores` | ユーザー訪問店舗 | ユーザーが訪問した店舗の記録（訪問日、メモなど） |
| `store_reports` | 店舗レポート | 店舗情報の報告・修正依頼（対応状況管理） |

## テーブル間の関係

### 主要なリレーション

- **users** ↔ **user_diagnose_results** ↔ **diagnose_results**
  - ユーザーは複数の診断結果を持つ
  - 診断結果は複数のユーザーに紐づく

- **users** ↔ **user_visited_stores** ↔ **stores**
  - ユーザーは複数の店舗を訪問記録として保存
  - 店舗は複数のユーザーに訪問される

- **diagnose_results** ↔ **diagnose_feedbacks**
  - 診断結果は1つのフィードバックを持つ（1対1）

- **stores** ↔ **store_reports**
  - 店舗は複数のレポート（報告）を持つ
